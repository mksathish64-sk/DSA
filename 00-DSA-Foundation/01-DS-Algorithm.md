# Data structure and Algorithm

## What is DSA?

DSA = Data Structure + Algorithm

## 1. What is Data Structure?

Data structure is a way of organizing and storing a data in a computer so that we can use it efficiently.

## 2. what is Algorithm?

Algorithm is a step by step procedure to solve a problem.

## 3. why do we need DSA?

Suppose we have only 10 numbers. Finding a number is easy.
What if we have 10 million numbers, if programs check one by one then it becomes slow.
using right **Data structure and Algorithm** makes the program:
- Faster
- Use less memory
- Handle large amount of data efficiently

>Note: " Good DSA helps program run faster and more efficiently "
